# 行動規範

## 私たちの約束

オープンで友好的なコミュニティを育てるために、コントリビューター・メンテナーとして、年齢・体型・障がい・民族・性自認と表現・経験レベル・国籍・外見・人種・宗教・性的指向にかかわらず、すべての人にとってハラスメントのないプロジェクト参加体験を提供することを約束します。

## 私たちの基準

ポジティブな環境を作るための行動例：

- 友好的で包括的な言葉を使う
- 異なる視点や経験を尊重する
- 建設的な批判を穏やかに受け入れる
- コミュニティにとって何が最善かを優先する
- 他のコミュニティメンバーへの共感を示す

容認できない行動例：

- 性的な言葉や画像の使用、および歓迎されない性的な注目やアプローチ
- 荒らし行為、侮辱・軽蔑的なコメント、個人攻撃・政治攻撃
- 公開または非公開のハラスメント
- 明示的な許可なく、住所や電子メールアドレスなど他者の個人情報を公開すること
- 職業上不適切と合理的に考えられるその他の行為

## 私たちの責任

プロジェクトのメンテナーは、許容される行動の基準を明確にする責任があり、容認できない行動に対して適切かつ公正な是正措置を取ることが期待されています。

プロジェクトのメンテナーは、この行動規範に沿わないコメント・コミット・コード・Wiki編集・Issue・その他のコントリビューションを削除・編集・拒否する権利と責任を持ち、不適切・脅迫的・攻撃的・有害と判断した行動を行ったコントリビューターを一時的または永久に追放することができます。

## 適用範囲

この行動規範は、プロジェクトスペース内、および個人がプロジェクトやそのコミュニティを公の場で代表する場合に適用されます。プロジェクトやコミュニティを代表する例として、公式プロジェクトのメールアドレスの使用、公式ソーシャルメディアアカウントによる投稿、オンライン・オフラインイベントでの代表者としての行動などがあります。

## 執行

虐待・ハラスメント・その他の容認できない行動を発見した場合は、プロジェクトチームに報告してください。すべての苦情は審査・調査され、状況に応じて必要かつ適切と判断される対応が取られます。プロジェクトチームは、インシデントの報告者に関する守秘義務を持ちます。
